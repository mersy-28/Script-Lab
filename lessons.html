Lesson 1: Introduction to the DOM

The DOM is a programming interface for web documents. It represents the page structure as a tree of nodes.

  • Access elements: getElementById, querySelector
  • Modify content: textContent, innerHTML
// HTML:

Lesson 5: Conditional Statements

      

        
        
      

      

        
Conditionals control program logic.

        
const score = 85;
if (score >= 90) {
  console.log("A");
} else if (score >= 80) {
  console.log("B");
} else {
  console.log("C or lower");
}
