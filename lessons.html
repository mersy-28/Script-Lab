Lesson 1: Introduction to the DOM

The DOM (Document Object Model) represents the structure of a web page as a tree of objects. It allows JavaScript to dynamically manipulate elements, styles, and attributes.

  • The DOM is structured as a tree of nodes.
  • You can use methods like document.getElementById() to select elements.
  • Events allow you to respond to user interactions with the page.

Lesson 2: Event Listeners

        

          
          
        

        

          
Event listeners are used to run JavaScript code in response to user interactions like clicks, key presses, or mouse movements.

          
    
            
  • Use element.addEventListener('event', callback) to attach a listener.
    • 
            
  • Common events include click, mouseover, and keydown.
    • 
            
  • You can use anonymous or named functions as callbacks.
